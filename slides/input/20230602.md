### RustとRubyの比較(言語仕様)

| Ruby |  | Rust |
| - | - | - |
| Dynamic(動的) | 型 | Static(静的) |

---

### RustとRubyの比較(開発環境)

| Ruby |  | Rust |
| - | - | - |
| Gem | ライブラリ | crate |
| .ruby-version(Dockerfile) + Gemfile | ライブラリの依存関係を明記するファイル | Cargo.toml |
| rubocop | LSP | rust-analyzer |
| rubocop | Formatter | rustfmt |
| rubocop | Linter | rustfmt |
| - | コンパイラ | rustc |
